Demo 1

Blijft de data behouden als je de pagina herlaadt?
Nee, de data wordt niet behouden.

Blijft de data behouden als je het tabblad sluit en de pagina opnieuw opent?
Nee, de data wordt niet behouden.

Gebruiken twee tabbladen die dezelfde pagina tonen, dezelfde data?
Nee, ze gebruiken beide verschillende data.

Demo 2

Blijft de data behouden als je het tabblad sluit en de pagina opnieuw opent?
Nee, De data wordt niet behouden.

Blijft de data behouden als je de pagina herlaadt?
Ja, de data wordt behouden.

Wordt de data gedeeld door verschillende tabbladen als je de pagina meermaals opent?
Nee, de data wordt niet gedeeld door verschillende tabbladen.

Denk je dat een andere pagina (andere file of url) aan de data geraakt als je ze in hetzelfde tabblad opent?
Nee, ik denk dat je niet aan de data graakt omdat het per sessie.

Demo 3

Blijft de data behouden als je het tabblad sluit en de pagina opnieuw opent?
Ja, de data wordt bijgehouden.

Blijft de data behouden als je de pagina herlaadt?
Ja, de data wordt bijgehouden.

Blijft de data behouden als je de browser afsluit, opnieuw start en de paginaweer opent?
Ja, de data wordt bijgehouden wanneer je uw browser afsluit en opnieuw opstart.

Blijft de data behouden als je de pagina in een andere browser opent?
Nee, de data wordt niet behouden in de andere browser.

Wordt de data gedeeld door verschillende tabbladen als je de pagina meermaals opent?
Ja, de data wordt gedeeld over de verschillende tabbladen.

Denk je dat een andere pagina (andere file of url) aan de data geraakt?
Ja, Denk van wel.
