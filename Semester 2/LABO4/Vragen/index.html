Antwoorden Labo 4

Antwoorden op de vragen over formulieren en strings

1. Waarom is het potentieel een probleem als je GET gebruikt in een situatie waar POST nodig is?

  • GET heeft een limiet op de hoeveelheid data.
  • Gegevens worden zichtbaar in de URL, wat een beveiligingsrisico kan zijn.
  • GET wordt gecached, wat ongewenste herhaling van acties kan veroorzaken.

2. Waarom moet een radiobutton een 'name' attribuut delen met andere radiobuttons in dezelfde groep?

Dit zorgt ervoor dat slechts één optie geselecteerd kan worden binnen dezelfde groep.

3. Wat is het verschil tussen <select> en <datalist>?

  • <select>: Biedt een vaste lijst met opties waaruit een gebruiker moet kiezen.
  • <datalist>: Laat de gebruiker een waarde intypen maar biedt suggesties.

4. Hoe kan je individuele karakters uit een string opvragen?

Gebruik .charAt(index), bijvoorbeeld: tekst.charAt(1) geeft het tweede karakter.

5. Hoe kan je in een string zoeken naar een deelwoord?

  • Gebruik .indexOf("woord") om te zoeken vanaf het begin.
  • Gebruik .lastIndexOf("woord") om te zoeken vanaf het einde.
